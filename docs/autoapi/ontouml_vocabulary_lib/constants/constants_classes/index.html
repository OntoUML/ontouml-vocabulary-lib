ontouml_vocabulary_lib.constants.constants_classes

This module defines various tuples of OntoUML Class Stereotypes used in OntoUML modeling.

Each tuple represents a specific grouping of stereotypes, providing a structured and semantically relevant means to access and utilize the stereotypes in OntoUML modeling.

All tuples are provided sorted in alphabetical order.
